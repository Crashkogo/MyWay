Заголовок самого последнего уровня

<div> - группирует или оборачивает другие элементы и семантически ничего не значит. Является блочным тэгом и сам по себе создаёт абзац.

<span> - строчный тэг, с его помощью можно выделить часть текста и наложить на него свои стили. При этом не разбивая и неперенося строку.

<hr> - горизонтальная черта разделяющая блоки на странице

<p> - создаёт блок, является абзацем. Всё что внутри него начнётся с новой строки. Имеет дефолтные стили для абзаца. Подробнее

<h1-h6> 6 уровней заголовков. Имеет тег открытия и закрытия, используется для более хорошей seo оптимизации на сайте.

<form> основной тэг форм, подробнее...

<input> это тэг поля ввода, включает в себя множество типов для разных задач ввода.

<texarea> - область ввода текста, можно задать размер поля. Имеет закрывающий тэг, можно установить текст по умолчанию.

<select> и внутри него <option> - поле для выбора из предопределенных значений. Тэг самого поля и значений соответственно.

<button> - обыкновенная кнопка

В уроке были показаны некоторые атрибуты, такие как: hiden, class, id, title

Кратко опишу для чего они. title - при наведении на объект отображает текст который мы обернули в этот атрибут. hidden - скрывает объект со страницы, имеет булево значение можно управлять JS скриптами. Но такое скрытие нельзя анимировать.

class - используется для css стилизации, одинаковый класс можно задать бесконечному количеству элементов на странице.

id - должен быть уникальным для каждого элемента. Позволяет создавать якори на странице.

Это не все глобальные атрибуты, тут можно почитать подробнее...
