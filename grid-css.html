Технология grid используется для дочерних элементов grid контейнера. Мы применяем свойство к верхнему элементу внутри которого лежат элементы которыми мы хотим управлять. HTML:
<div class="grid-container">
<div class="item">
<div class="item">
<div class="item">
</div>
CSS:
.grid-container{
display: grid;
}
Линия сетки. Линии бывают как линии колонки так и линии строки. Линия идёт слева на право или сверху вниз и отсчёт начинается с 1.

тут мы указываем свойства display: grid;
grid-template-colums: 2fr 1fr 1fr;
fr - это фракция свободного пространства. Тут мы получаем 1 столбец на 2 фракции и 2 столбца по 1, что наглядно видно. Фракции занимают весь объём контейнера.
Фракции можно использовать десятичные 0.5fr и т.п.
Так же укажем тут свойство grid-column-gap: 10px, что является отступом между колонками шириной в 10 пикселей.
grid-row-gap: 20px; Тоже самое только для строк.

тут мы указываем свойства display: grid;
grid-template-colums: repeat(2,0.5fr 2fr );
repeat() функция повторения. Конкретно тут мы используеся двойное повторение двух значений фракций
Так же можно использовать функцию minmax() внутри функции повторения. например если указать repeat(3,minmax(150px,1fr)), тогда при изменении размера окна браузера минимальный размер ячейки будет 150 пикселей, а максимальый 1 занимаемая фракция.

При использовании grid area задаём имена областям внутри основного контейнера. Например:
.item1-area{grid-area: header}
После создания имён, мы можем использовать свойство:
grid-templates-areas:
"header header header header"
"main main main sidebar"
"footer footer footer sidebar"
На примере видно, что мы как бы "графически" изображаем область как распологаются наши дочернии контейнеры которым мы дали названия.
Далее добавим медиа запрос к нашей области.
@media screen and (max-width: 900px){
.grid-container-area{
grid-template-rows: 150px 150px 300px 50px;
grid-template-areas:
"header header header header"
"sidebar sidebar sidebar sidebar"
"main main main main"
"footer footer footer footer";
}
}
