CSS-позиционирование позволяет управлять расположением элементов на веб-странице. Ключевое свойство для позиционирования — это position, и в зависимости от его значения поведение элемента меняется. Рассмотрим три важных значения: static, relative, и absolute.
Свойство position
Свойство position задает метод позиционирования элемента на странице. Оно определяет, как элемент будет взаимодействовать с другими элементами и как будут применяться дополнительные свойства позиционирования (top, left, right, bottom).
static - Это значение по умолчанию для всех элементов. Элементы с position: static располагаются в естественном потоке документа (одно за другим), без особого позиционирования.
Свойства top, left, right, bottom не работают с этим значением.
.box {
position: static;
width: 200px;
height: 100px;
}
Элемент будет отображен в нормальном порядке потока, не изменяя своего положения.
relative (относительное позиционирование):
Элемент с position: relative остается в потоке документа, но его можно смещать относительно собственного исходного положения, используя свойства top, left, right, и bottom.
Свойства top, left, right, bottom смещают элемент от его начальной позиции, не затрагивая других элементов, оставшихся на своих местах.
box1
box2
.box {
position: relative;
top: 20px; /* Смещение элемента вниз на 20 пикселей */
left: 10px; /* Смещение элемента вправо на 10 пикселей */
}
absolute (абсолютное позиционирование):
Описание: Элемент с position: absolute удаляется из обычного потока документа и позиционируется относительно ближайшего предка с position: relative, absolute или fixed. Если таких предков нет, он позиционируется относительно страницы (краев окна).
Свойства top, left, right, bottom задают точное положение элемента относительно его содержащего блока.

fixed (фиксированное позиционирование):
Описание: Элемент с position: fixed фиксируется относительно окна браузера и остается в одном и том же месте при прокрутке страницы.
Свойства top, left, right, bottom устанавливают фиксированное положение элемента внутри окна.
Пример на новой странице тут.

right: 0;
top: 0;
left: 0;
}
 Так же в этом примере важно отодвинуть всю часть контента на размер фиксируемого блока.

}
sticky (липкое позиционирование):
Описание: Элемент с position: sticky ведет себя как relative до тех пор, пока не пересечет определенный порог (например, верхнюю границу окна), после чего становится фиксированным на месте.
Свойства top, left, right, bottom указывают точку, в которой элемент "прилипает" к странице.
Пример на новой странице тут.
.menu2{
position: sticky;
top: 0;
}

Функция calc() используется для выполнения математических вычислений прямо в CSS, чтобы задавать значения свойств. Она позволяет комбинировать разные единицы измерения (px, %, em, vw и т.д.) в одном выражении. 

.content { 
   padding: calc(10px + 2%); /* Отступ = 10px плюс 2% ширины контейнера */ 
 }
Используем calc() там, где нужно учитывать динамические размеры (например, гибкая верстка).
