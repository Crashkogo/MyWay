Название таблицы
Первая ячейка таблицы а тут вторая                   

Тэг <table> основной для создания таблицы (открывающий/закрывающий) тут в примере для отображения границ было использовано свойство border="1" у этого тэга.

Тэг <tr> это строка таблицы, а <td> элемент таблицы образующий столбец внутри строки. Оба тэга открываются и закрываются.

Для названия используется тэг <caption>

div - блочный тэг создаёт абзац, позволяет включать в блок другие тэги для разметки по странице. span - строчный тэг hr - тэг сплошной линии. p - обычный абзац, имеет встроенный отступ сверху и снизу. h1-h6 - заголовки, могут быть от 1 до 6, у каждого разные отступы по умолчанию. Является семантическим тэгом имеет смысл при чтении браузером
